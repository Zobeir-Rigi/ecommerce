# E-Commerce UI Design

A responsive e-commerce application built with React and Styled Components. The project focuses on creating a simple and intuitive shopping experience, allowing users to browse products, manage a shopping cart, and complete a checkout flow.

## Live Demo

https://frontend-practice-ecommerce.netlify.app/

## Features

- Browse products through a responsive user interface
- Add and remove items from the shopping cart
- Checkout functionality
- Responsive design for desktop, tablet, and mobile devices
- State management using React Hooks
- Reusable React components

## Tech Stack

- React
- JavaScript
- Styled Components
- HTML5
- CSS3

## Project Overview

This project was built to strengthen my understanding of React and modern frontend development practices. It demonstrates component-based architecture, state management using React Hooks, and responsive UI development using Styled Components.

## Installation

Clone the repository:

```bash
git clone <https://github.com/Zobeir-Rigi/ecommerce.git>
```

Navigate to the project directory:

```bash
cd ecommerce
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```

Open the application at:

```text
http://localhost:3000
```
